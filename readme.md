## Deployment

To start CRA

```bash
  npm run start
```

## Production

To deploy this project run

```bash
  npm run build
```

## Features

- Css Loader project
- SVG Loader project
- Optimization with webpack 5
- Image Loader project
- Different settings on changing the development environment

## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
  npm run start
```
